# 机械结构设计说明

## 连杆参数

| 部件 | 尺寸（mm） | 说明 |
|------|-----------|------|
| 连杆1（大臂） | 125 × 15 × 3 | 肩关节 → 肘关节，中心距 125mm |
| 连杆2（小臂） | 125 × 12 × 3 | 肘关节 → 末端，中心距 125mm |
| 底座板 | 60 × 40 × 3 | 固定舵机1，螺丝固定到桌面 |

> 中心距 = 舵机轴孔中心到下一关节轴孔中心的距离，连杆两端孔径 Ø4mm（M3 螺丝间隙配合）。

## 零件清单

- 底座固定板 × 1
- 连杆1 × 1
- 连杆2 × 1
- MG90S 舵机臂（原配十字臂） × 2
- M2×8 自攻螺丝 × 4（固定舵机臂）
- M3×10 螺丝 + M3 螺母 × 4（关节铰链）
- 橡皮垫 / 防滑脚垫 × 4（底座）

## 3D 打印推荐参数

- 材料：PLA 或 PETG
- 层厚：0.2mm
- 填充：25%（蜂窝或三角形）
- 外壳层数：3
- 支撑：无需（平放打印）

> 轻量化建议：在连杆中间镂空椭圆形减重孔（保留 5mm 边框即可），可将每根连杆从 \~8g 减至 \~4g。

## 装配顺序

1. 将舵机1 固定在底座板上（4 个 M2 螺丝）
2. 安装舵机1 的原配舵机臂，与连杆1 用 M3 螺丝连接
3. 将舵机2 固定在连杆1 末端的舵机座槽中
4. 安装舵机2 的舵机臂，与连杆2 用 M3 螺丝连接
5. 末端根据用途安装笔夹 / 吸嘴 / 激光座

## 轴向说明

```
俯视图（水平面）：

    BASE
     │
     ○ ← 舵机1 轴（向上穿出桌面）
     │
     ╔═══════════════╗  ← 连杆1（L1=125mm）
              │
              ○ ← 舵机2 轴（向上）
              │
              ╔══════════╗  ← 连杆2（L2=125mm）
                        │
                       [末端]
```

两个舵机轴均**垂直于水平桌面**，整个机械臂在水平面内运动，重力方向与运动方向垂直，舵机无需克服重力负载。

## 接线图

```
外部 5V 2A 电源
    ├── VCC ──→ 舵机1 红线
    │         舵机2 红线
    └── GND ──→ 舵机1 棕线
               舵机2 棕线
               Arduino GND

Arduino Nano
    D9  ──→ 舵机1 橙线（信号）
    D10 ──→ 舵机2 橙线（信号）
    USB  ← → PC 串口通信 + 供电 Arduino
```

**注意**：不要直接用 Arduino 5V 引脚给舵机供电，Arduino 板载稳压器限流 \~500mA，两个舵机同时动作可能超过此限制导致复位。
